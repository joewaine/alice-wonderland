# Alice in Wonderland

A browser-based 3D platformer collectathon set in Lewis Carroll's Alice in Wonderland.

Built with Three.js + Rapier.js + TypeScript.

## Play

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Space | Jump |
| Q | Shrink |
| R | Grow |
| E | Talk to NPCs |
| Arrow Keys | Camera |
| ESC | Pause |

## Features

- 4 playable chapters from Alice in Wonderland
- Size-changing mechanic for puzzle solving
- NPC dialogue with authentic book quotes
- Collectibles: Golden Keys, Wonder Stars, Playing Cards
- Procedural audio - no external files needed
- Particle effects for visual feedback

## Tech Stack

- **Vite** - Build tool
- **Three.js** - 3D rendering
- **Rapier.js** - Physics engine
- **TypeScript** - Type safety
- **Web Audio API** - Procedural sound effects
